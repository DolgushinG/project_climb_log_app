E2E Test Runner

Запуск E2E тестов против localhost (Playwright)

Откроется страница GitHub Actions. Нажмите «Run workflow» для запуска.

Локальный запуск:

flutter build web --dart-define=USE_DEV_API=true --dart-define=E2E_MODE=true --no-web-resources-cdn --web-renderer html
npx serve build/web -p 8080 &
cd e2e && npm install && npx playwright test
