# Описание обновления для RuStore

Краткий текст «Что нового» для карточки версии в RuStore (можно сократить под лимит поля).

---

## Полный вариант (для «Описание обновления»)

- **Гостевой режим** — просмотр рейтинга и соревнований без входа; вкладка «Войти» для перехода в аккаунт.
- **Вход по Passkey** — вход в аккаунт по Face ID / Touch ID (биометрия).
- **Улучшена работа при слабом интернете** — корректная обработка сетевых ошибок, офлайн-очередь и уведомления.
- **Обновлён профиль и аналитика** — доработан экран профиля и отображение статистики.
- **Участие в соревнованиях** — обновлён процесс записи: авто-категории, наборы зачётов, экран чекаута и история участий.
- **Рейтинг** — новый экран рейтинга (RatingScreen), интеграция в экран соревнований и главное меню.
- **Пуш-уведомления RuStore** — поддержка уведомлений через RuStore (при подключении бэкенда).
- **Исправления и доработки интерфейса** — экраны соревнований, результатов и «О приложении».

---

## Короткий вариант (до ~500 символов)

Гостевой режим: рейтинг и соревнования без входа. Вход по Passkey (Face ID / Touch ID). Улучшена работа при слабом интернете и обработка ошибок сети. Обновлены профиль и аналитика. Добавлен экран рейтинга. Доработана запись на соревнования: авто-категории, наборы зачётов, чекаут. Поддержка пуш-уведомлений RuStore. Исправления и улучшения интерфейса.

---

## Минимальный вариант (1–2 строки)

Гостевой режим без входа, вход по биометрии (Passkey), экран рейтинга, улучшения при слабом интернете, обновлённый профиль и запись на соревнования. Пуш-уведомления RuStore и исправления.
