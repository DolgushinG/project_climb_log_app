# Интеграция замеров в план и standalone-режим

## Текущее состояние

- **Экран Тест**: замеры (пальцы, щипок, тяга, lock-off) — сохранение через API.
- **План**: ОФП/СФП дни, календарь, выполнение упражнений.
- **Связь**: генератор плана по замерам убран с экрана Тест.

## Цели

### 1. План + замеры

Когда у пользователя есть план:
- Встроить дни замеров в календарь (например, после каждых 2–3 СФП).
- На экране дня плана: если день = замер, показывать форму замеров вместо упражнений.
- Результаты замеров сохраняются и используются для адаптации плана (бэкенд).

### 2. Без плана: только замеры

Когда плана нет:
- Пользователь ведёт замеры вручную через экран Тест.
- История замеров доступна (StrengthHistoryScreen, Прогресс).
- Рейтинг «Топ недели» показывает участников с показателями.

### 3. Когда делать замеры

**Рекомендация: после 2–3 СФП-сессий.**

- СФП даёт нагрузку на пальцы; замеры на «свежих» пальцах искажают результат.
- После 2–3 СФП организм адаптировался, но ещё не перетренирован.
- Альтернатива: раз в 1–2 недели фиксированный день замеров (например, воскресенье после отдыха).

**Интервал между замерами:** минимум 3–5 дней, оптимально 7–14 дней для отслеживания прогресса.
