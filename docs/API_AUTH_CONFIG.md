# API Auth Config — feature flags для входа по соцсетям

Flutter загружает конфиг при открытии экрана входа и показывает кнопки только для тех соцсетей, для которых флаг `true`.

## Endpoint

```
GET /api/auth/config
```

**Публичный** — не требует авторизации.

## Ответ

```json
{
  "social_login": {
    "vkontakte": true,
    "telegram": true,
    "yandex": false
  }
}
```

- `vkontakte` — показывать кнопку «Войти через VK»
- `telegram` — показывать кнопку «Войти через Telegram»
- `yandex` — показывать кнопку «Войти через Yandex»

Если флаг `false` или ключ отсутствует — кнопка не показывается.

## Поведение Flutter

- При ошибке/таймауте/404 — все кнопки соцсетей скрыты
- Если `social_login` не объект — скрыты
- Раздел «Или войти с помощью» показывается только если хотя бы один флаг `true`
