ПРОМПТ ДЛЯ БЭКЕНДА: Статистика соревнования

Контекст: в мобильном приложении есть раздел «Детали соревнования» с вкладками «Информация», «Результаты», «Статистика». Сейчас вкладка «Статистика» пустая. Нужно добавить общую сводку по соревнованию.

ЧТО НУЖНО РЕАЛИЗОВАТЬ

Новый эндпоинт: GET /api/event/{event_id}/statistics

Возвращает JSON:
{
  "participants_total": 87,
  "participants_male": 52,
  "participants_female": 35,
  "routes_total": 15,
  "flashes_total": 124,
  "redpoints_total": 89,
  "zones_total": 45
}

Описание полей:
- participants_total — общее кол-во участников
- participants_male — кол-во мужчин (gender = male / Мужской)
- participants_female — кол-во женщин (gender = female / Женский)
- routes_total — общее кол-во трасс (квалификация + полуфинал + финал)
- flashes_total — сумма флешей по всем участникам (attempt = 1 в результатах)
- redpoints_total — сумма редпоинтов (attempt = 2)
- zones_total — сумма зон (attempt = 3)

Источники данных:
- Участники: таблица участников события (там есть gender)
- Трассы: трассы события
- Результаты: таблица результатов (route_id, user_id, attempt), где attempt: 0=X, 1=Flash, 2=Redpoint, 3=Zone
