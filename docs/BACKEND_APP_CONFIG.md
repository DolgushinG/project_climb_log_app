# Backend API: конфиг приложения и фича-флаги

## GET /api/app/config

Загружается при входе пользователя в приложение (MainScreen). Позволяет менять поведение без обновления апп.

**Авторизация:** Bearer token обязателен.

**Ответ 200:**
```json
{
  "features": {
    "ai_coach_enabled": true
  }
}
```

Или на верхнем уровне:
```json
{
  "ai_coach_enabled": true
}
```

### Поля

| Поле | Тип | Описание |
|------|-----|----------|
| `features.ai_coach_enabled` | boolean | Вкладка «AI Тренер» в разделе Тренировки. `false` — скрыть. По умолчанию `true` при отсутствии поля. |

### Ошибки

- `401` — Flutter использует закэшированный конфиг или fallback.
- `404` / ошибка сети — fallback (все фичи включены).

### Кэш

При каждом заходе в приложение запрашивается свежий конфиг (forceRefresh). Кэш используется только как fallback при ошибке сети. При logout кэш очищается.
